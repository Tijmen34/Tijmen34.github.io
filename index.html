Progress

In this small section, I will show how far I am with each competency, as well as showing evidence for every competency.

Professional Craftmanship

Professional Craftmanship is a competency which I feel I have been working on continuously throughout my study. Whenever I take a class, I learn a bit more about becoming a professional Software Engineer. If I were to reflect on how far I am progressing with working on this competency, I would say I would be at about 70%.

70%

Evidence for my Professional Craftmanship would be:

  • Code from project FYS
  • Reflection from project PAD
  • PAD Product

The code from my FYS project shows how much I had learned after only one semester of programming, while the reflection from PAD shows how I reflected on my wrong decision regarding a piece of code I had to create. After reflecting on it, I decided to make new code for me proces re-examination which ended up being far better. I wish I was able to show code for project PAD, unfortunately it is all made in Mendix, which is a visual programming IDE.

That is why I am giving myself a 45% for this competency aswell.

45%

Evidence for my Learning Ability would be:

Currently I need to work on this competency a lot. I need to do more with the old feedback I got from my previous project and make goals for the next project based on that feedback.
